# CLAUDE.md

## Commit messages

Prefix every commit subject with a conventional-commit type:

- `feat:` — new functionality
- `fix:` — bug fix
- `docs:` — documentation-only changes
- `refactor:` — code change that doesn't add a feature or fix a bug
- `chore:` — tooling, deps, config, other maintenance

Example: `fix: use wildcard safe.directory since chart dir is a repo subdirectory`
